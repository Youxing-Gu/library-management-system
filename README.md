# 图书角管理系统 作业提交

## 一、项目概述
本项目为“图书角”管理系统的UML建模与版本控制实践，完成了用例图、类图的设计，并通过Git进行版本管理，推送到GitHub公开仓库。

## 二、UML 模型设计
### 1. 用例图
- **角色**：普通用户、管理员
- **核心用例**：图书搜索、借阅图书、归还图书、查询个人借阅记录、图书信息录入、图书信息维护、借阅记录管理、用户信息管理
- **关系**：所有操作均包含「登录」前置用例，符合系统安全逻辑

### 2. 类图
- **核心类**：`Book`（图书）、`User`（用户）、`BorrowRecord`（借阅记录）
- **关联关系**：用户与借阅记录为1对多，图书与借阅记录为1对多，符合业务逻辑
